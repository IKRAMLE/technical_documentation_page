HTML Basics

HTML stands for HyperText Markup Language.

HTML is used to structure content on the web.

HTML elements are the building blocks of HTML.

<div> <p> <h1>
  • HTML uses tags to describe elements.
  • Tags are enclosed in angle brackets.
  • The closing tag has a slash (/) before the tag name.

CSS Fundamentals

CSS stands for Cascading Style Sheets.

CSS is used to style the layout of web pages.

CSS can be applied to HTML elements in different ways.

color: blue; font-size: 16px; background-color: lightgray;
  • CSS can be inline, internal, or external.
  • CSS selectors define which elements to style.
  • Properties define the appearance of the selected element.

JavaScript Introduction

JavaScript is a programming language used to create dynamic and interactive effects on web pages.

It runs in the browser and can modify the HTML content and CSS styles.

JavaScript is a key element of the web's functionality.

let x = 10; document.getElementById('example'); console.log("Hello, World!");
  • JavaScript can manipulate the DOM (Document Object Model).
  • JavaScript is event-driven.
  • Variables in JavaScript can store values of different types.

Responsive Design

Responsive Design makes web pages look good on all devices by adjusting layout based on screen size.

Media queries in CSS are used to implement responsive design.

Responsive design ensures that users have an optimal viewing experience.

@media (max-width: 768px) {} width: 100%; height: auto;
  • Responsive design uses flexible grid layouts.

  • Responsive images scale based on the container size.
